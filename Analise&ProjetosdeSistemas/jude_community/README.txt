                                 README
                             JUDE/Community
=========================================================================

【概要】

  JUDEは、Java 言語とUML1.4によるオブジェクト指向分析設計などのモデリングを
  サポートするツールです。

【機能・特徴】
  JUDE/Communityは、以下のような機能を持ちます。
   ・UML1.4全図サポート
   ・クラス図では、オブジェクト図、パッケージ図、ロバストネス図もサポート
   ・Java ソースファイルからモデル情報を生成
   ・モデル情報から Java のスケルトンコードを生成
   ・モデル情報からクラス図の自動生成
   ・BMP形式による他ツールへの貼り付け

  特に、以下のような特徴があります。
   ・新たなダイアログを毎回開かずに図を直接編集可能
   ・右ドラッグでの自由なスクロール
   ・全体マップで大きな図内を自在に移動
   ・マウスドラッグによるズーム
   ・無限回の UNDO/REDO
   ・多彩なレイアウト調整機能(位置、サイズ揃え)

【動作環境】

  J2SE1.4.1_03以降 がインストールされていることが必要です。
 （Sun Microsystems以外のJava環境上の動作は、動作保証対象外となります。 ）
  Windows 2000とWindows XP、Windows Vistaにて動作を確認しています。

【インストール・起動方法】

  1. J2SEのインストール
    推奨されるJ2SEがインストールされていない場合は、あらかじめインストールします。
    J2SEのインストール方法は、J2SEのホームページを参照ください。
    http://java.sun.com/j2se/1.4.2/ja/download.html
    【動作環境】にあるとおり、J2SE1.4.1_03以降をインストールしてください。
    （例 j2re-1_4_2_12-windows-i586-p.exe）

  2. JUDEのインストール
    古いバージョンがインストールされている場合は、あらかじめアンインストールしてください。

    ◎インストーラ利用の場合
      ダウンロードしたインストーラ（jude_xxx-setup.exe)を実行します。
      ※Javaの特別なオプション設定などが必要な場合は、jude.batを編集してください。
      ※Windows98、MEの場合は、jude.batを編集して、JUDE_HOMEを設定してください

    ◎インストーラを利用しない場合
      ダウンロードしたアーカイブ（jude_xxx.zip)を適当なフォルダに解凍します。

  3. 起動

    ◎起動方法１(インストーラ使用の場合)
    スタートアップメニュー、または、デスクトップのショートカットから起動します。
    また、judeファイルをダブルクリックして、JUDEを起動します。

    ◎起動方法２
    Windowsユーザで、J2REがインストールされている場合は、jude-community.jarを
    ダブルクリックするだけで起動します。

    jarのダブルクリックで起動できない場合は、以下の方法で起動します。

    ◎起動方法３
    Windowsユーザは、アーカイブに含まれるバッチファイル(jude.bat)を利用できます。
    (自分の環境にあわせて編集してください)。

    ◎起動方法４
    コマンドラインから以下のように起動します。
    > java -Xmx256m -Xss2m -jar jude-community.jar

  ※Export HTML を利用する場合は、J2SE JDKに含まれるtools.jarを、
    jude-community.jarと同じフォルダにコピーしてください。 

    うまく起動できない場合、jude-info@change-vision.com にお問合わせ下さい。

【アンインストール】
  インストーラでインストールした場合：
    1. スタートメニューから"アンインストール JUDE Community"を実行してください。

  インストーラを利用していない場合：
    1. jude-community.jar を削除してください。
    2. コピーしたtools.jarを削除してください。

【古いモデルに関するご注意】

    以前のバージョン（特にJude竹）で作成されたモデルを編集する場合は、
    一旦新しいバージョンのJUDEで開き、[ツール]-[モデルの補正]-[不正なモデルを補正]
    を実行してから、お使いいただきますようお願いいたします。
    これを実行しない場合は、編集時に問題が発生する場合があります。
